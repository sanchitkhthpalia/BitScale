# BitScale

Responsive data grid component built with React and Vite based on Figma design.

## Features

- Responsive data grid
- Search and filter functionality
- Smooth animations and interactions

## Tech Stack

- React 19.2.3 with TypeScript
- Vite 6.2.0
- Lucide React icons

## Getting Started

```bash
git clone https://github.com/sanchitkhthpalia/BitScale_Assgn.git
cd BitScale_Assgn
npm install
npm run dev
```

## Deployment

**Live Demo**: [Deployed Application Link](https://bitscale-gilt.vercel.app/)

**Platform**: Vercel  
**Build**: `npm run build`  
**Output**: `dist`
